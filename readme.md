# To-Do List API (Mongoose + Express)

This project is a REST API for managing todo items, built with **Node.js**, **Express**, and **Mongoose**.

## Features

- **Create Todo:** `POST /` with `{ "kegiatan": "Your activity" }`
- **Read Todos:** `GET /` returns all items.
- **Update Todo:** `PUT /:id` with `{ "kegiatan": "...", "status": "..." }`
- **Delete Todo:** `DELETE /:id`

## Tech Stack

- **Server:** Express.js
- **Database:** MongoDB (via Mongoose)
- **Middleware:** cors, morgan, body-parser

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start server:
   ```bash
   npm start
   ```
